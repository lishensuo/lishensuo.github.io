1、XGBoost#

  • boosting算法是按顺序构建一系列弱学习器模型。每一个新建的模型都在试图改正前序模型的错误。根据纠正前序模型方式的不同，可分为自适应boosting（代表的是Adaboost）与梯度boosting（代表的是XGBoost）

Adaboost

（1）训练第一个模型，采用自助法抽样时，所有训练集样本具有相同权重，即每个样本被抽到的概率相同。

（2）使用训练好的第一个模型进行全部样本做出预测，对于分类正确的样本，降低其权重；对于分类错误的样本提高其权重。如此更新所有样本的权重。

（3）训练第二个模型，采用自助法抽样时，根据样本最新的权重进行抽样，所以之前分类错误的样本更容易被抽到。即第二个模型可针对第一个模型的分类错误样本进行训练。

（4）此时有两个模型：根据模型准确度，对每个模型赋予一个预测权重。然后根据每个样本在两个模型的预测结果的权重和，做出最终预测。如果预测正确，降低其权重；预测错误，提高其权重。如此更新所有样本的权重。

（5）训练第三个模型，采用自助法抽样时，根据样本最新的权重进行抽样，所以之前分类错误的样本更容易被抽到。即第三个模型可针对前序模型(第一与第二个模型)的分类错误样本进行训练。

（6）此时有三个模型：根据模型准确度，对每个模型赋予一个预测权重。然后根据每个样本在三个模型的预测结果的权重和，做出最终预测。如果预测正确，降低其权重；预测错误，提高其权重。如此更新所有样本的权重。

….不断重复（3）、（4）两个步骤，直至达到预定义的子模型数量或者达到某个预定的准确度，形成最终的集成模型

  • 关于 XGBoost算法还不太理解，涉及到梯度 Gradient，很多公式。有一个视频觉得很不错，之后再学

task_classif = as_task_classif(Zoo2, target = "type")
task_classif$col_roles$stratum = "type"

learner = lrn("classif.xgboost", predict_type="prob")
learner$param_set
  • eta：学习率，取值范围在0~1之间。值越大，学习越快，也容易过拟合；值越小，花费时间就越长
  • gamma：类似随机森林的cp超参数
  • max_depth：允许最大深度
  • min_child_weight：划分节点的最小杂质程度
  • subsample：每个决策树的随机抽样的样本比例
  • colsample_bytree：每个决策树采样预测变量的比例
  • nround：顺序构建的决策树数量
  • eval_metric：损失函数类型

search_space = ps(
  eta = p_dbl(lower=0,upper=1),
  gamma = p_dbl(lower=1,upper=5),
  max_depth = p_int(lower = 1, upper = 10),
  min_child_weight = p_dbl(lower = 1, upper = 10),
  subsample = p_dbl(lower = 0.5, upper = 1),
  colsample_bytree = p_dbl(lower = 0.5, upper = 1),
  nrounds = p_int(lower = 1, upper = 50),
  eval_metric = p_fct(c("merror","mlogloss"))
)
design = expand.grid(eta = c(0.1, 0.5),
                     gamma = c(1, 3),
                     max_depth=c(4, 8),
                     min_child_weight = c(3, 6),
                     subsample = c(0.6, 0.9),
                     colsample_bytree = c(0.6, 0.9),
                     nrounds = c(10, 20),
                     eval_metric = c("merror","mlogloss"),
                     stringsAsFactors = F) %>% as.data.table()

resampling = rsmp("cv")
resampling$param_set$values$folds = 5
measure = msr("classif.acc")

instance = TuningInstanceSingleCrit$new(
  task = task_classif,
  learner = learner,
  resampling = resampling,
  measure = measure,
  terminator = trm("none"),
  search_space = search_space
)

future::plan("multisession")
tuner$optimize(instance)
