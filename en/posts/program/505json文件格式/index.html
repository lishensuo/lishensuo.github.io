  • JSON (JavaScript Object Notation)：一种轻量级的数据交换格式，易于人类阅读和编写，同时也易于机器解析和生成。它广泛用于网络数据传输和配置文件。
  • JSON的整体是由花括号{}定义的键值对，值包括字符串、数字、布尔值、对象、数组和 null

    • null → Python None
