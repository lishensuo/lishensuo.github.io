Adobe Illustrator快速入门

Create: 2025-10-04 | Update: 2025-10-04 | Words: 693 | 2 min | Lishensuo | Viewers:

Adobe Illustrator (AI)

  • 矢量图软件：和 Photoshop（位图）不同，AI 绘制的图形放大不会失真。

  • 对象化思维：你画的每个图形（矩形、箭头、线条、文字）都是独立对象，可以随时调整大小、颜色、线条粗细。

  • 图层管理：科研图往往有多层（背景、形状、文字、标注），合理用图层能让修改更轻松。

路径是由 锚点 (Anchor Point) 和 连接锚点的线段（直线或曲线） 组成的对象。可以填充颜色（闭合路径）或只显示描边.

锚点是路径上的 关键点，定义路径的形状和方向。

  • 角点 (Corner) → 尖锐折角

  • 曲率点 / 平滑点 (Smooth) → 连续平滑

快捷键

  • 移动画布：空格键 + 鼠标左击
  • 缩放画布：Alt + 鼠标滚轮
  • 移动/缩放对象：V （选择工具）
    • 将整个对象当成一个整体来移动、缩放、旋转。
  • 调整形状某个锚点/边：A （直接选择工具）
    • 点一下矩形的角 → 只选中这个角，可以单独拖动，变成梯形
    • 点一条曲线 → 可以调整贝塞尔曲线控制点，改变曲线弯曲程度
  • 绘制矩形/圆形工具：M / L （矩形/椭圆工具）
  • 绘制直线：\ （直线段工具）
  • 绘制刚性线段/形状：P （钢笔工具）
    • 在路径上点击：创建锚点
    • 在锚点上点击：删除锚点
    • 按Esc / V 退出
  • 绘制曲线：Shift + ~ （曲率工具）
  • 创建文字：T （文字工具）

置入pdf绘图：File → Place… 或者快捷键 Ctrl + Shift + P

导入pdf后的颜色与原有pdf颜色存在差异

  • Illustrator 文档有 RGB 和 CMYK 两种颜色模式：
    • File → Document Color Mode → RGB / CMYK
  • 如果 PDF 是 RGB，当前 AI 文档是 CMYK → 导入时颜色会自动转换，可能偏暗或偏淡

存储为 (Save As)

  • AI（Illustrator 文件） → 完全可编辑、保留图层、锚点、文字属性
  • PDF（可选择保留 Illustrator 功能） → 可编辑矢量，但有兼容性选项
  • SVG → 矢量网页格式，可保留可编辑属性

导出为（Export As）

  • PNG / JPEG / TIFF → 位图，用于论文、PPT
